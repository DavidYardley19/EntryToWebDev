What is the purpose of the for attribute?

Another way to associate an input element with labels is to utilise the "for" attribute.
This means that the input element does not necessarily need to be within the label element.
This is false information, find out the real info below:
You can plant the text within a label element, ensuring that the for attribute is associated with the identifier attribute in the input option.
